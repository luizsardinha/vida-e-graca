# Tema Vida & Graça (v1.0.0)

Este é o tema oficial da **Revista Vida & Graça**, desenvolvido com foco em estética editorial moderna, performance e acessibilidade.

## 🚀 Como instalar

1.  Comprima a pasta `vida-e-graca-theme` em um arquivo `.zip`.
2.  No seu WordPress (v6.9.1), vá em **Aparência > Temas > Adicionar Novo > Enviar Tema**.
3.  Faça o upload do arquivo `.zip` e clique em **Ativar**.

## 📊 Como importar os dados de exemplo (Mock Data)

Para que o site não comece vazio, fornecemos um arquivo de importação com **6 artigos editoriais reais**:

1.  Vá em **Ferramentas > Importar** no seu painel WordPress.
2.  Instale e execute o **Importador do WordPress**.
3.  Selecione o arquivo `mock-data.xml` localizado na raiz da pasta do tema.
4.  Siga as instruções para atribuir autores e importar anexos.

**Conteúdo incluído:**
- Editorial de boas-vindas (Cassiano Bento).
- Entrevista The Send (Paulo Borges Jr).
- Memória Haroldo Walker.
- Marcha pra Jesus 2026.
- Esporte e Fé (Rayssa Leal).
- Valores Masculinos (Hamilton Jr).

## 🌓 Modo Claro e Escuro

O tema detecta automaticamente a preferência do sistema do usuário, mas permite a troca manual através do botão de sol/lua no cabeçalho. A escolha é salva no navegador.

## 🛠 Desenvolvimento (Tailwind CSS)

Se precisar fazer alterações no design:
1.  Certifique-se de ter o Node.js instalado.
2.  Execute `npm install` na pasta do tema.
3.  Use `npm run watch` para desenvolvimento em tempo real ou `npm run build` para gerar o arquivo de produção final.

## ♿ Acessibilidade

-   **Skip Links:** Pressione `TAB` logo após carregar a página para ver a opção de pular direto para o conteúdo.
-   **Navegação Semântica:** Uso rigoroso de tags HTML5 (`<article>`, `<header>`, `<main>`, `<nav>`).
-   **Contraste:** Paleta de cores otimizada para legibilidade.
